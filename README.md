## How to build slides

* Install Scala >3.5.0 from here: https://www.scala-lang.org/download/

* Install texlive from here: https://www.tug.org/texlive/

* Run this command in terminal:

```
scala run .
```

* Slides are created in the `target` dir in file `slides.pdf`
